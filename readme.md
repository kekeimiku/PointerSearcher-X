# PointerSearcher-X

> Memory dynamic pointer chain search tool.

## Support:

- [x] aarch64-darwin

- [x] aarch64-apple-ios

- [x] x86_64-linux-gnu

- [x] aarch64-linux-gnu

- [x] aarch64-linux-android (beta)

- [ ] x86_64-darwin (alpha)

- [ ] x86_64-windows (alpha)

- [ ] nintendo-switch
