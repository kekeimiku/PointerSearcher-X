# PointerSearcher-X

> Memory dynamic pointer chain search tool.

# 2026-03-06

The PINCE plugin remains free to use, and its binary is auditable.

Other targets will use license activation.

A new version with better performance and compatibility will be released later.

## Support:

- [x] aarch64-macos

- [x] aarch64-apple-ios

- [x] x86_64-linux-gnu

- [x] aarch64-linux-gnu

- [x] aarch64-linux-android (beta)

- [x] x86_64-macos (alpha)

- [ ] x86_64-windows (alpha)

- [ ] nintendo-switch
